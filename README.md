# Sea-Salon
 Seasalon adalah sebuah website yang menyediakan layanan pemesanan dan ulasan salon kecantikan. 
## Instalasi

1. **Clone repository**:
    ```bash
    git clone https://github.com/yourusername/sea-salon.git
    cd sea-salon
    ```

2. **Instal dependensi**:
    ```bash
    npm install
    ```

3. **Jalankan server pengembangan**:
    ```bash
    npm start
    ```
